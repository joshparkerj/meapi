What is this?

The Mass Effect API, or "meapi" (Mee-pee) is the world's foremost quantified and programmatically-accessible data source for all the data from the Mass Effect series!

I've taken all the rich contextual stuff from the universe and formatted it into something that software can easily consume. You can access everything I've got!

How can I use it?

For now, you're just going to have to wing it! Documentation probably forthcoming at some point.

Try using postman, it's a great tool.
